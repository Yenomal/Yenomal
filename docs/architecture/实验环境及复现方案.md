# 实验环境及复现方案

## 实验环境说明

- 在实际复现过程中，环境是最为重要的，其包括：

1. 系统环境：系统使用的各种C++文件版本，这一部分需要通过docker等容器方案解决

2. 代码环境：复现时使用的代码，这一部分需要回退到对应的git tag位置解决

3. python环境：复现时使用的import依赖，这一部分需要uv实现

## 复现过程

- 针对具体的环境，复现需要复现出代码当时过程：

1. 系统环境目前通过conda+人工配置bashrc实现，但是这样的方式在开源的时候必然需要修改成docker

2. 代码环境目前通过git tag + manifest.sh脚本一键实现submodule+主repo的复原

3. python环境的uv实现非常复杂，此处引出一个 [pyproject.toml](../template/pyproject.toml) 根据实际需要配置相关参数
